## Relevant Universal Truths to honour

These are the ones that specifically apply to this build and that we cannot skip:

- **#13** — Server-render discoverable content. No SPA. No client-only rendering for the canonical page.
- **#26** — Same, reinforced. If content needs to be indexed, it must be in the initial HTML response.
- **#33** — WCAG AA minimum, respect `prefers-reduced-motion`, focus states visible
- **#58** — Responsive images with `srcset` and `sizes`
- **#80** — Scale is a system. Use tokens throughout (10/20/40/60/80 spacing, `--text-*`, `--radius-*`).
- **#82** — Don't localise syntax (British English in prose, US English in CSS keys — this is Marbl's established pattern)
- **#91** — Component library CSS shouldn't force `!important` on consumers
- **#92** — Mechanical constraints must precede judgment review (hooks, linters, CI checks before Moirai)
